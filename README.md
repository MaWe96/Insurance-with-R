# Insurance-with-R

### Paket
`tidyverse`

### Filer
_insurance_costs.csv_, _notebook.Rmd_, _report.pdf_.

### Köra projektet
1. Ladda ner repo, ha .csv, scripts och .Rmd i samma mapp.
2. Kör alla kodblock

**Notebooken utför:**
- Läser in data, städar, sätter datatyper och fyller i blanka värden med `median` eller `unknown`.
- Skapar variablerna åldersgrupp, BMI-klass, historik, risk och träning+kontroller.
- Skapar visualiser.
- Genomför t-test, regressionsmodeller och felanalys.
